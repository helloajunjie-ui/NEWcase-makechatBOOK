# 🎭 尼可剧本工具 · V2.1.0

> **作者：尼可** · **QQ 群：1051068329**

> **格式转换 · AI 铸造 · AI 动态生成沉浸式 HTML 宣发页 · 灵感对话 — 一站式剧本管理平台**

春潮 / 风月 / MISS 剧本格式互转，AI 辅助创作，AI 根据世界观动态生成主题 HTML 宣发页，内置 AI 灵感对话助手，全部在浏览器本地完成。

## ✨ 核心功能

### 📐 格式转换器

**一次解析，任意导出。** 拖入 JSON 文件，自动识别来源格式，一键导出为 Markdown / 春潮 / 风月 / MISS。

- **单文件模式**：拖拽或粘贴 JSON，实时预览转换结果
- **批量模式**：同时处理多个文件，一键打包 ZIP 下载
- **无损转换**：所有字段完整保留，跨平台往返一致
- **UID 去重**：每个剧本绑定确定性 UID，同名同源文件重复导入自动覆盖更新，避免数据冗余

### 📚 剧本库

**浏览器本地存储（IndexedDB），刷新不丢失。** 所有解析过的剧本自动入库。

- **卡片列表**：标题、摘要、标签、来源、🌐 宣发页状态一目了然
- **详情弹窗**：完整查看提示词、世界书、宣发页（预览/下载/查看源码/重新铸造）
- **宣发页源码编辑器**：已铸造宣发页可点击「📄 查看源码」打开弹窗，支持：
  - **查看/编辑**：可编辑的 HTML 源码 textarea，等宽字体，语法高亮选中
  - **一键复制**：📋 按钮复制完整源码到剪贴板
  - **格式化**：🔧 自动缩进整理 HTML 结构
  - **保存修改**：💾 保存后自动入库并刷新详情面板
- **一键复制**：每条提示词、每条世界书都带 📋 复制按钮
- **右键菜单**：快速复制、导出、删除
- **搜索**：按标题快速定位剧本

### 🏭 AI 剧本工坊

**从零开始，用 AI 铸造全新剧本。** 4 步向导，从种子词到完整剧本。

| 步骤 | 做什么 | 耗时 |
|------|--------|------|
| 💡 **脑暴抽卡** | 输入种子词/梗概，AI 生成 3 张概念卡 | ~10s |
| ✏️ **精修确认** | 编辑、合并、删除卡片，确定最终设定；可导入已有素材（捏人/世界观/体系），设定玩家扮演角色 | 手动 |
| 📖 **提示词膨胀** | AI 将短大纲扩展为 800+ 字结构化 Prompt（含主角身份） | ~15s |
| 🌍 **世界书 & 宣发** | AI 生成世界书条目；支持追加指令迭代扩充；AI 铸造专属宣发页；保存入库 | ~10s |

> 需要自行配置 AI API Key（右上角 🤖），预设 DeepSeek 端点。

**流水线集成：** 捏人/世界观/体系 三个独立工坊的产出，可在 Step 2 一键导入作为剧本素材，AI 在后续步骤中自动融合这些设定。

### 🧑 AI 捏人工坊

**一句话描述 → AI 生成完整人物档案。** 独立于 4 步工坊的轻量角色生成器，快速产出有血有肉的角色设定。

- **输入**：一句话描述人物特点（如"废土独眼机械师，表面冷漠内心温柔"）
- **输出**：结构化人物档案，包含姓名/年龄/性别/外貌/身份/身世/性格/能力/口头禅/背景故事
- **一键复制**：格式化的完整档案文本，直接粘贴到剧本中使用
- **流水线导出**：生成结果自动保存，可在剧本工坊 Step 2 导入作为素材

### 🌍 AI 世界观构建

**一句话设定 → AI 构建完整世界观档案。** 独立于捏人的世界设定生成器，快速搭建有内在逻辑的世界框架。

- **输入**：一句话世界设定（如"灵气复苏的赛博朋克世界，科技与修仙并存"）
- **通用字段**：世界名称/标签/时代背景/地理环境/核心法则 — 骨架级信息始终展示
- **补充内容**：主要势力/种族/科技体系/冲突危机/标志场景等 — AI 输出的丰富细节折叠收纳，点击展开
- **流水线导出**：生成结果自动保存，可在剧本工坊 Step 2 导入作为素材

### ⚡ AI 体系工坊

**一句话设定 → AI 构建完整的力量体系/升级框架。** 独立于世界观和人物的体系设计器，快速产出有逻辑自洽性的能力框架。

- **输入**：一句话描述体系设定（如"九阶修仙体系，每阶分初期/中期/后期/圆满，突破需渡劫"）
- **通用字段**：体系名称/标签/类型/等级划分/核心机制 — 骨架级信息始终展示
- **补充内容**：各等级详解/特殊分支/变异路线/克制关系等 — 折叠收纳，点击展开
- **适用场景**：修仙等阶、异能升级树、功法体系、血脉进化、装备品级等
- **流水线导出**：生成结果自动保存，可在剧本工坊 Step 2 导入作为素材

### 💬 灵感对话

**与 AI 进行多轮灵感对话，辅助剧本创作。** 内置多轮记忆管理，自动持久化，不丢失上下文。

- **多轮记忆**：自动保存最近 20 轮对话（滑动窗口），超出自动裁剪
- **无感存档**：每次对话后自动持久化到 `localStorage`，刷新页面不丢失
- **TXT 导出**：一键导出完整对话历史为格式化文本文件
- **清空对话**：一键重置，开始全新灵感碰撞

### 🌐 AI 动态生成沉浸式 HTML 宣发页

**每个剧本都值得一个独一无二的高逼格独立入口。** 这不是简单的模板套壳——AI 根据剧本的世界观、风格、标签，从零手写一套匹配题材的 HTML/CSS/JS 单页应用。

- **自动提取**：导入剧本时，源 JSON 中已有的 HTML 宣发页自动入库（春潮 `detailIntro` / MISS `description`），直接预览/下载
- **AI 动态铸造**：没有宣发页？AI 根据剧本题材实时生成沉浸式 HTML 页面：
  - 🦾 **赛博朋克** → 霓虹发光 + 故障艺术 CSS + 暗黑矩阵背景
  - 🏔️ **修仙武侠** → 水墨晕染 + 古风排版 + 毛笔字体
  - 🌌 **奇幻科幻** → 深色渐变 + 粒子背景 + 金色装饰边框
  - 🧛 **哥特暗黑** → 暗红黑金 + 哥特字体 + 浮雕纹理
  - 🏖️ **日常现代** → 柔和渐变 + 毛玻璃效果 + 圆润卡片
- **主角身份**：生成宣发页时自动读取 Step 2 设定的玩家扮演角色，注入页面
- **预览/下载**：浏览器内 Blob 秒级预览，一键下载独立 HTML 文件
- **查看/编辑源码**：点击「📄 查看源码」打开弹窗，可编辑 HTML、一键复制、格式化缩进、保存修改
- **重新铸造**：不满意随时重新生成，新版本覆盖入库

### 🎨 樱花动态背景

**沉浸式 WebGL 樱花飘落特效。** 800 个 3D 粒子，带景深模糊和辉光后处理，`z-index: -1` 不干扰任何交互操作。

- 通过 JS 动态创建 Canvas，零 HTML 侵入
- 性能优化：粒子数从 1600 减至 800，不阻塞主线程
- 所有视图通用，自动适配明暗主题

## 🚀 快速上手

### 1. 打开工具

直接双击 [`index.html`](./index.html) 在浏览器中打开，**零依赖，无需安装**。
> 推荐使用 `npx http-server -p 8080 --cors` 启动本地服务器以获得最佳体验。

### 2. 转换剧本

1. 切换到 **📐 转换器** 视图
2. 拖拽 JSON 文件到上传区域（或粘贴 JSON 内容）
3. 选择输出格式 → 复制或下载

### 3. 管理剧本库

1. 切换到 **📚 剧本库** 视图
2. 点击左侧卡片查看详情
3. 右键卡片快速操作
4. 卡片上的 🌐 徽章表示已有宣发页，点击可直接预览

### 4. AI 创作

1. 点击右上角 **🤖** 配置 AI（填入 API Key）
2. 切换到 **🏭 剧本工坊** 视图
3. 按 4 步向导逐步操作
4. 在 Step 2 可导入捏人/世界观/体系素材，设定玩家扮演角色
5. 在 Step 4 可追加世界书内容，生成宣发页

### 5. AI 动态生成宣发页

- **已有宣发页**：导入后详情页直接显示 ✅ 已铸造，点击预览/下载
- **AI 铸造**：在详情页点击 **✨ AI 铸造专属主题宣发页**，AI 根据世界观实时生成匹配题材的 HTML 页面
- **预览**：点击 👁 预览，浏览器内 Blob 内存秒开新标签页展示
- **下载**：点击 ⬇ 下载，保存为独立 HTML 文件
- **查看/编辑源码**：点击 **📄 查看源码**，在弹窗中编辑 HTML、一键复制、格式化、保存修改
- **重新铸造**：点击 🔄 重新铸造，AI 重新生成覆盖入库

### 6. 灵感对话

1. 切换到 **💬 灵感对话** 视图
2. 在输入框输入你的灵感或问题，按回车或点击发送
3. AI 回复后自动保存对话历史
4. 点击 **📥 导出 TXT** 下载完整对话记录
5. 点击 **🗑 清空对话** 重置会话

### 7. 独立工坊

- **🧑 捏人**：一句话 → AI 生成完整人物档案 → 复制/导入流水线
- **🌍 世界观**：一句话 → AI 构建世界观 → 复制/导入流水线
- **⚡ 体系**：一句话 → AI 设计力量体系 → 复制/导入流水线

## ⚠️ 注意事项

- **所有数据在浏览器本地处理**，不上传任何内容到服务器
- AI 功能需要自行配置 API Key，Key 仅存储在本地 localStorage
- 封面/背景图为外部链接，可能随源站变动而失效
- 宣发页 HTML 存储在 IndexedDB 中，导出时需手动下载
- 同名同源 JSON 文件重复导入会自动覆盖更新（基于 UID 去重），不会产生重复条目
- 灵感对话记录存储在 localStorage，清除浏览器数据会丢失
