# 🎭 酒馆剧本工坊

> **作者：尼可** · **QQ 群：1051068329**

> **格式转换 · AI 铸造 · AI 动态生成沉浸式 HTML 宣发页 — 一站式剧本管理平台**

春潮 / 风月 / MISS 剧本格式互转，AI 辅助创作，AI 根据世界观动态生成主题 HTML 宣发页，全部在浏览器本地完成。

---

## ✨ 核心功能

### 📐 格式转换器

**一次解析，任意导出。** 拖入 JSON 文件，自动识别来源格式，一键导出为 Markdown / 春潮 / 风月 / MISS。

- **单文件模式**：拖拽或粘贴 JSON，实时预览转换结果
- **批量模式**：同时处理多个文件，一键打包 ZIP 下载
- **无损转换**：所有字段完整保留，跨平台往返一致

### 📚 剧本库

**浏览器本地存储，刷新不丢失。** 所有解析过的剧本自动入库。

- **卡片列表**：标题、摘要、标签、来源一目了然
- **详情弹窗**：完整查看提示词、世界书、宣发页
- **一键复制**：每条提示词、每条世界书都带 📋 复制按钮
- **右键菜单**：快速复制、导出、删除
- **搜索**：按标题快速定位剧本

### 🏭 AI 剧本工坊

**从零开始，用 AI 铸造全新剧本。** 4 步向导，从种子词到完整剧本。

| 步骤 | 做什么 | 耗时 |
|------|--------|------|
| 💡 **脑暴抽卡** | 输入种子词/梗概，AI 生成 3 张概念卡 | ~10s |
| ✏️ **精修确认** | 编辑、合并、删除卡片，确定最终设定 | 手动 |
| 📖 **提示词膨胀** | AI 将短大纲扩展为 800+ 字结构化 Prompt | ~15s |
| 🌍 **世界书衍生** | AI 生成世界书条目，写入本地文件 | ~10s |

> 需要自行配置 AI API Key（右上角 ⚙️），预设 DeepSeek 端点。

### 🌐 AI 动态生成沉浸式 HTML 宣发页

**每个剧本都值得一个独一无二的高逼格独立入口。** 这不是简单的模板套壳——AI 根据剧本的世界观、风格、标签，从零手写一套匹配题材的 HTML/CSS/JS 单页应用。

- **自动提取**：导入剧本时，源 JSON 中已有的 HTML 宣发页自动入库，直接预览/下载
- **AI 动态铸造**：没有宣发页？AI 根据剧本题材实时生成沉浸式 HTML 页面：
  - 🦾 **赛博朋克** → 霓虹发光 + 故障艺术 CSS + 暗黑矩阵背景
  - 🏔️ **修仙武侠** → 水墨晕染 + 古风排版 + 毛笔字体
  - 🌌 **奇幻科幻** → 深色渐变 + 粒子背景 + 金色装饰边框
  - 🧛 **哥特暗黑** → 暗红黑金 + 哥特字体 + 浮雕纹理
  - 🏖️ **日常现代** → 柔和渐变 + 毛玻璃效果 + 圆润卡片
- **预览/下载**：浏览器内 Blob 秒级预览，一键下载独立 HTML 文件
- **重新铸造**：不满意随时重新生成，新版本覆盖入库

## 🚀 快速上手

### 1. 打开工具

直接双击 [`index.html`](./index.html) 在浏览器中打开，**零依赖，无需安装**。

### 2. 转换剧本

1. 切换到 **📐 转换器** 视图
2. 拖拽 JSON 文件到上传区域（或粘贴 JSON 内容）
3. 选择输出格式 → 复制或下载

### 3. 管理剧本库

1. 切换到 **📚 剧本库** 视图
2. 点击左侧卡片查看详情
3. 右键卡片快速操作

### 4. AI 创作

1. 点击右上角 **⚙️** 配置 AI（填入 API Key）
2. 切换到 **🏭 剧本工坊** 视图
3. 按 4 步向导逐步操作

### 5. AI 动态生成宣发页

- **已有宣发页**：导入后详情页直接显示 ✅ 已铸造，点击预览/下载
- **AI 铸造**：在详情页点击 **✨ AI 铸造专属主题宣发页**，AI 根据世界观实时生成匹配题材的 HTML 页面
- **预览**：点击 👁 预览，浏览器内 Blob 内存秒开新标签页展示
- **下载**：点击 ⬇ 下载，保存为独立 HTML 文件
- **重新铸造**：点击 🔄 重新铸造，AI 重新生成覆盖入库

## ⚠️ 注意事项

- **所有数据在浏览器本地处理**，不上传任何内容到服务器
- AI 功能需要自行配置 API Key，Key 仅存储在本地
- 封面/背景图为外部链接，可能随源站变动而失效
- 宣发页 HTML 存储在 IndexedDB 中，导出时需手动下载
